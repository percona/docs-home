# Percona XtraDB Cluster 8.4.2-2 has been released

<!-- more -->

[Percona XtraDB Cluster 8.4.2-2](https://docs.percona.com/percona-xtradb-cluster/8.4/){.md-button} has been released on Wednesday, December 11, 2024.
