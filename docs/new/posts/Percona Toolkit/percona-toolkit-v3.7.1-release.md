# Percona Toolkit v3.7.1 has been released

<!-- more -->

[Percona Toolkit v3.7.1](https://docs.percona.com/percona-toolkit/){:target="_blank"} has been released on Wednesday, December 17, 2025.

This version has the following new features:

* Adds -skip-pod-summary for pt-k8s-debug-collector

* Adds SSL options to DSN

* Adds invisible option to pt-duplicate-key-checker

* Allows pt-stalk to disable ps-lock-transactions data collection via parameter

* Adds support for Debian 13

* Adds support for Amazon Linux 2023

Learn more in the Percona Toolkit v3.7.1 [release notes](https://docs.percona.com/percona-toolkit/release_notes.html#v3-7-1-released-2025-12-17){:target="_blank"}.
