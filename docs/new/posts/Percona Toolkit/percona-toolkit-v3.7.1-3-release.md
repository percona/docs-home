# Percona Toolkit v3.7.1-3 has been released

<!-- more -->

[Percona Toolkit v3.7.1-3](https://docs.percona.com/percona-toolkit/){:target="_blank"} has been released on Friday, April 17, 2026.

This release addresses multiple important security vulnerabilities affecting `libarchive` (CVE-2026-4424, CVE-2026-4111), `libnghttp2` (CVE-2026-27135), `gnupg2` (CVE-2025-68973), and `openssl-libs` (CVE-2025-15467).

Learn more in the Percona Toolkit v3.7.1-3 [release notes](https://docs.percona.com/percona-toolkit/release_notes.html#v3-7-1-3-released-2026-04-17){:target="_blank"}.
