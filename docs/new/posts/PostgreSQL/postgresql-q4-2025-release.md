# New releases of Percona Distribution for PostgreSQL: 17.7, 16.11, 15.15, 14.20, 13.23 (EOL) have been published

<!-- more -->

The following releases of [Percona Distribution for PostgreSQL](https://docs.percona.com/postgresql/17/index.html){:target="_blank"} were published in November-December 2025:

* [17.7](https://docs.percona.com/postgresql/17/release-notes/release-notes-v17.7.1.html){:target="_blank"}
* [16.11](https://docs.percona.com/postgresql/16/release-notes/release-notes-v16.11.html){:target="_blank"}
* [15.15](https://docs.percona.com/postgresql/15/release-notes-v15.15.html){:target="_blank"}
* [14.20](https://docs.percona.com/postgresql/14/release-notes/release-notes-v14.20.html){:target="_blank"}
* [13.23 (EOL)](https://docs.percona.com/postgresql/13/release-notes/release-notes-v13.23.html){:target="_blank"}

These releases follow version 18.1 and complete the PostgreSQL major version release cycle for Q4 2025.

Try out Percona Distribution for PostgreSQL using the [Quickstart guide](https://docs.percona.com/postgresql/17/installing.html){:target="_blank"}.
