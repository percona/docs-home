# Percona Distribution for PostgreSQL 17.10.1 , 16.14, 15.18, and 14.23 have been released

<!-- more -->

This distribution has new releases for the following versions, published in May 2026:

* [17.10](https://docs.percona.com/postgresql/17/release-notes/release-notes-v17.10.1.html){:target="_blank"}
* [16.14](https://docs.percona.com/postgresql/16/release-notes/release-notes-v16.14.html){:target="_blank"}
* [15.18](https://docs.percona.com/postgresql/15/release-notes-v15.18.html){:target="_blank"}
* [14.23](https://docs.percona.com/postgresql/14/release-notes/release-notes-v14.23.html){:target="_blank"}

These releases follow version 18.4 and complete the PostgreSQL major version release cycle for Q2 2026.

Try out Percona Distribution for PostgreSQL using the [Quickstart guide](https://docs.percona.com/postgresql/18/quick-start.html){:target="_blank"}.
