# Percona Distribution for PostgreSQL 18.4.2, 17.10.2 and `pg_tde` 2.2.1 have been released

<!-- more -->

We're pleased to announce the release of [Percona Distribution for PostgreSQL](https://docs.percona.com/postgresql/18/index.html){:target="_blank"} 18.4.2, 17.10.2 and [`pg_tde`](https://docs.percona.com/pg-tde/index.html){:target="_blank"} 2.2.1!

This is an out-of-cycle maintenance release addressing high-severity CVEs where fixes were available.

This release covers the following:

- Percona Distribution for PostgreSQL 18.4.2 and 17.10.2 are now available.
- `pg_tde` has been updated to version 2.2.1, including stability and compatibility improvements.
- PostGIS has been updated to version 3.5.7, pgVector to 0.8.3, and pgpool to 4.7.2 for PostgreSQL 14-18.
- Docker images have been rebuilt with updated components and fixes for high-severity CVEs, where available.
- Upgrade images have been updated.
- pgBackRest image has been updated with high-severity CVE fixes.
- pgBouncer image has been updated with high-severity CVE fixes.

!!! note
    PostgreSQL 14–16 do not have new Percona Distribution releases in this update cycle. Updated packages for PostGIS, pgVector, and pgpool, along with refreshed Docker images, are available for these versions.

For more details, see the following release notes:

- [18.4.2 release notes](https://docs.percona.com/postgresql/18/release-notes/release-notes-v18.4.2.html)
- [17.10.2 release notes](https://docs.percona.com/postgresql/17/release-notes/release-notes-v17.10.2.html)
- [pg_tde 2.2.1 release notes](https://docs.percona.com/pg-tde/release-notes/release-notes-v2.2.1.html)
