# Percona Distribution for PostgreSQL 17.9.1, 16.13, 15.17, and 14.22 have been released

<!-- more -->

This distribution has new releases for the following versions, published in November-December 2025:

* [17.9](https://docs.percona.com/postgresql/17/release-notes/release-notes-v17.9.1.html){:target="_blank"}
* [16.13](https://docs.percona.com/postgresql/16/release-notes/release-notes-v16.13.html){:target="_blank"}
* [15.17](https://docs.percona.com/postgresql/15/release-notes-v15.17.html){:target="_blank"}
* [14.22](https://docs.percona.com/postgresql/14/release-notes/release-notes-v14.22.html){:target="_blank"}

These releases follow version 18.3 and complete the PostgreSQL major version release cycle for Q1 2026.

Try out Percona Distribution for PostgreSQL using the [Quickstart guide](https://docs.percona.com/postgresql/18/installing.html){:target="_blank"}.
