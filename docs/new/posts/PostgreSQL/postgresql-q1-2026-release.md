# Percona Distribution for PostgreSQL 17.8, 16.12, 15.16, and 14.21 have been released

<!-- more -->

This distribution has new releases for the following versions, published in November-December 2025:

* [17.8](https://docs.percona.com/postgresql/18/release-notes/release-notes-v17.8.1.html){:target="_blank"}
* [16.12](https://docs.percona.com/postgresql/16/release-notes/release-notes-v16.11.html){:target="_blank"}
* [15.16](https://docs.percona.com/postgresql/15/release-notes-v15.15.html){:target="_blank"}
* [14.21](https://docs.percona.com/postgresql/14/release-notes/release-notes-v14.20.html){:target="_blank"}

These releases follow version 18.1 and complete the PostgreSQL major version release cycle for Q4 2025.

Try out Percona Distribution for PostgreSQL using the [Quickstart guide](https://docs.percona.com/postgresql/17/installing.html){:target="_blank"}.
