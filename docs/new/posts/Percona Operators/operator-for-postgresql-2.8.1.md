# Percona Operator for PostgreSQL 2.8.1 has been released

<!-- more -->

[Percona Operator for PostgreSQL](https://docs.percona.com/percona-operator-for-postgresql/2.0/){:target="_blank"} 2.8.1 has been released on December 16, 2025.

Try it out using the [Quickstart guide](https://docs.percona.com/percona-operator-for-postgresql/2.0/quickstart.html){:target="_blank"}.

New features and improvements introduced by this release include:

* Support for PostgreSQL 18. This latest major version of PostgreSQL delivers major improvements in performance, usability, and security, empowering you to make large-scale, mission-critical deployments more reliable and efficient. Read more about PostgreSQL 18 in:

    * [Percona Blog: Planning Ahead for PostgreSQL 18: What Matters for Your Organization :octicons-link-external-16:](https://www.percona.com/blog/planning-ahead-for-postgresql-18-what-matters-for-your-organization/)
    * [PostgreSQL 18.1 release notes :octicons-link-external-16:](https://www.postgresql.org/docs/18/release-18-1.html)

    Find PostgreSQL 18 images in the list of [Percona-certified images](https://docs.percona.com/percona-operator-for-postgresql/latest/images.html){:target="_blank"}.

Learn more in Percona Operator for PostgreSQL 2.8.1 [release notes](https://docs.percona.com/percona-operator-for-postgresql/2.0/ReleaseNotes/Kubernetes-Operator-for-PostgreSQL-RN2.8.1.html){:target="_blank"}.
