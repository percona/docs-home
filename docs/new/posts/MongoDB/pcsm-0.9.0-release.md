# Percona ClusterSync for MongoDB 0.9.0 has been released

<!-- more -->

[Percona ClusterSync for MongoDB](https://docs.percona.com/percona-clustersync-for-mongodb/index.html){:target="_blank"} 0.9.0 has been released on Jun 03, 2026. 

Try it out using the [Quickstart guide](https://docs.percona.com/percona-clustersync-for-mongodb/installation.html){:target="_blank"} 

This release introduces cross-version replication support, allowing replication from MongoDB 6.0 and 7.0 source clusters to MongoDB 8.0 targets. This capability helps simplify staged upgrades and migration workflows while maintaining continuous replication between supported MongoDB versions.

We also ship a Software Bill of Materials (SBOM) with every release, providing visibility into the components and dependencies included in each release.

Learn more about other changes in Percona ClusterSync for MongoDB 0.9.0 [release notes](https://docs.percona.com/percona-clustersync-for-mongodb/release-notes/0.9.0.html){:target="_blank"}.
