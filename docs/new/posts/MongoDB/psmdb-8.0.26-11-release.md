# Percona Server for MongoDB 8.0.26-11, 7.0.37-20, and 6.0.29-23 are now available

<!-- more -->

We are pleased to announce the availability of the following releases of Percona Server for MongoDB:

- **8.0.26-11** — released on **June 25, 2025**
- **7.0.37-20** — released on **June 23, 2025**
- **6.0.29-23** — released on **June 24, 2025**

These releases include bug fixes, security updates, and improvements that enhance the stability and reliability of Percona Server for MongoDB.

## Get started

Install or upgrade using the Quickstart Guide for your version:

- [Percona Server for MongoDB 8.0 Quickstart](https://docs.percona.com/percona-server-for-mongodb/8.0/install/index.html){:target="_blank"}
- [Percona Server for MongoDB 7.0 Quickstart](https://docs.percona.com/percona-server-for-mongodb/7.0/install/index.html){:target="_blank"}
- [Percona Server for MongoDB 6.0 Quickstart](https://docs.percona.com/percona-server-for-mongodb/6.0/install/index.html){:target="_blank"}
