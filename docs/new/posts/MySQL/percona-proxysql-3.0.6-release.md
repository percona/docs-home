# Percona Build of ProxySQL 3.0.6 has been released

<!-- more -->

[ProxySQL](https://docs.percona.com/proxysql/){:target="_blank"} 3.0.6 has been released on Thursday, April 16, 2026.

Try it out using the [Installation guide](https://docs.percona.com/proxysql/install-v3.html){:target="_blank"}.

This version updates the Percona Build of ProxySQL to version 3.0.6. The ProxySQL Admin tool and the pxc_scheduler_handler tool within Percona's Build of ProxySQL 3.0.6 have not been updated and are the same as in the previous Percona release.

Learn more in ProxySQL 3.0.6 [release notes](https://docs.percona.com/proxysql/3.0.6.html){:target="_blank"}.
